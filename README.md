# Advent of code

Repository for advent of code solutions.

## Running Go Solutions

Run `main.go` in the root folder by piping the required problem input. For example:
`go run main.go <year> <day> < <input-file.txt>`

## Running the Web Server

The project includes a web interface served by the Go application. To run it:
`go run main.go -web`
This will start the web server accessible at `http://localhost:8080`
